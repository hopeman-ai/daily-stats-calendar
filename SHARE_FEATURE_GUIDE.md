# 공유 기능 사용 가이드

## ✅ 현재 상태

**공유 기능이 이미 완벽하게 구현되어 있습니다!**

따뜻한 한 문장 생성 + 공유 기능이 모두 작동합니다.

## 🎯 사용 방법

### 1. 데이터 입력
1. 브라우저에서 `index.html` 열기
2. 원하는 날짜 클릭
3. 다음 정보 입력:
   - 에너지 상태 (높음/보통/낮음)
   - 하루 태그 1-2개 (예: #집중 #휴식)
   - 메모 (선택)
4. "저장" 버튼 클릭

### 2. 따뜻한 한 문장 확인
저장 버튼을 누르면 자동으로 **"오늘의 한 문장"** 모달이 표시됩니다.

예시:
- "요즘 계속 힘든 시간이었던 걸 생각하면, 오늘도 쉽지 않았을 것 같아. 그래도 여기까지 온 걸로 충분해."
- "최근 며칠을 보면, 오늘도 너 나름대로 하루를 채운 것 같아."

### 3. 공유하기

모달에 두 개의 버튼이 있습니다:

#### 📋 문장 복사
- 클릭하면 **문장만 클립보드에 복사**됩니다
- 복사 후 원하는 곳(카카오톡, 인스타그램, 메모장 등)에 붙여넣기 가능
- 알림: "문장이 복사되었습니다"

**사용 예:**
```
복사된 문장을 다음에 붙여넣기:
- 카카오톡 나와의 채팅방
- 인스타그램 스토리 텍스트
- 노션/옵시디언 일기
- 트위터/X 포스트
```

#### 📤 원하면 공유
- 클릭하면 **이미지 카드가 자동 생성**됩니다
- 이미지에는 다음이 포함됩니다:
  - ✅ 오늘의 한 문장 (중앙 정렬)
  - ✅ "요즘 어때?" 서비스명 (하단)
  - ❌ 날짜 정보 (개인정보 보호)
  - ❌ 위치 정보

**모바일에서:**
- Web Share API 지원 시: 공유 팝업이 자동으로 뜹니다
  - 인스타그램, 카카오톡, 메시지 등 선택 가능
- 미지원 시: 이미지가 자동 다운로드됩니다

**PC에서:**
- 이미지가 자동 다운로드됩니다 (Downloads 폴더)
- 파일명: `daily-summary.png`
- 다운로드 후 직접 원하는 곳에 업로드

### 4. 저장된 요약 다시 보기
- 날짜를 다시 클릭하면 저장된 요약 문장이 모달에 표시됩니다
- "오늘의 한 문장" 섹션에서 확인 가능

## 🎨 생성되는 이미지 특징

### 현재 버전 (기본)
- **크기**: 800x600px
- **배경**: 흰색
- **카드**: 회색 배경 (#f5f5f5)
- **텍스트**:
  - 요약 문장 (28px, Segoe UI)
  - "요즘 어때?" (18px, 하단)

### 포함 정보
- ✅ 오늘의 따뜻한 한 문장
- ✅ 서비스명
- ❌ 날짜 (개인정보 보호)
- ❌ 에너지 상태
- ❌ 태그
- ❌ 위치 정보

## 📱 플랫폼별 사용 팁

### 인스타그램 스토리
1. "📤 원하면 공유" 클릭
2. 인스타그램 선택 (모바일) 또는 이미지 저장 후 업로드 (PC)
3. 스티커, 배경음악 등 자유롭게 편집

### 카카오톡
1. "📋 문장 복사" 클릭
2. 나와의 채팅방에 붙여넣기
3. 또는 "📤 원하면 공유" → 카카오톡 선택

### 노션/옵시디언
1. "📋 문장 복사" 클릭
2. 일기 페이지에 붙여넣기
3. 또는 "📤 원하면 공유" → 이미지 다운로드 → 페이지에 삽입

### 트위터/X
1. "📋 문장 복사" 클릭
2. 새 트윗에 붙여넣기
3. 또는 "📤 원하면 공유" → 이미지 첨부

## 🔧 문제 해결

### "복사 버튼이 작동하지 않아요"
- **원인**: 브라우저 권한 설정
- **해결**:
  - 브라우저에서 클립보드 권한 허용
  - 또는 문장을 드래그해서 수동 복사

### "공유 버튼을 눌렀는데 아무 일도 안 일어나요"
- **원인**: Web Share API 미지원 또는 팝업 차단
- **해결**:
  - 팝업 허용 설정
  - Downloads 폴더에서 이미지 확인
  - 파일 탐색기 또는 Finder에서 `daily-summary.png` 찾기

### "이미지에 한글이 깨져요"
- **원인**: 폰트 렌더링 문제
- **해결**:
  - 대부분의 경우 문제없음
  - 브라우저 캐시 초기화 (Ctrl + Shift + Delete)

### "모바일에서 공유 팝업이 안 떠요"
- **원인**: 구형 브라우저 또는 iOS 제한
- **해결**:
  - 이미지가 자동 다운로드됨
  - 사진 앱에서 찾아서 공유

## 💡 활용 아이디어

### 1. 일일 감정 기록
- 매일 저녁 5분 투자
- 따뜻한 문장으로 하루 마무리
- 노션/옵시디언에 모아서 보관

### 2. SNS 정기 포스팅
- 주 1회 요약 이미지 공유
- "#요즘어때" 해시태그와 함께
- 팔로워들과 일상 나누기

### 3. 친구/가족과 공유
- 힘든 날 문장 복사해서 전송
- 서로 응원하는 메시지로 활용

### 4. 개인 아카이브
- 매월 말 한 달치 문장 정리
- 나의 감정 변화 추적
- 연말 회고 자료로 활용
